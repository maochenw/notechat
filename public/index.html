Frequently Asked Questions

How does the self-destructing chat room work?

When you create a room, a countdown timer starts. Once it reaches zero, the room and all messages are permanently destroyed. Nothing is saved to any server or database — messages only exist in real-time while the room is active.

Do I need an account or login?

No. Chat Off Record requires no sign-up, no email, and no login. Just pick a display name and start chatting. Your identity stays anonymous.

Can anyone read my messages?

No. We do not store, log, or monitor any message content. Messages are transmitted in real-time between participants and are never written to a database. Once the room is destroyed, the conversation is gone permanently.

What is a permanent private room?

For a one-time payment, you can create a room with a custom URL that doesn't expire. You can optionally protect it with a codeword. Messages still disappear when everyone leaves — only the room link persists.
